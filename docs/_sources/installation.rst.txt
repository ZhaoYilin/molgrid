############
Installation
############

Prerequisites
=============

- Python 3.8 or higher
- NumPy 1.16.0 or higher

Installing MolGrid
===================

From Source
-----------

Clone the repository and install in development mode:

.. code-block:: bash

    git clone https://github.com/yourusername/molgrid.git
    cd molgrid
    pip install -e .

With Dependencies
------------------

To install MolGrid with all optional dependencies for development and testing:

.. code-block:: bash

    # Installation with test framework
    pip install -e ".[test]"

    # Installation with Sphinx documentation tools
    pip install -e ".[docs]"

    # Installation with all extras
    pip install -e ".[all]"

Verifying Installation
=======================

To verify the installation, you can run:

.. code-block:: python

    import molgrid
    print(molgrid.__version__)

Or run the test suite:

.. code-block:: bash

    pytest test/

Requirements
============

Core Dependencies:

- **NumPy** (>=1.16.0): Numerical computing library for array operations and linear algebra

Optional Dependencies:

- **pytest** (>=7.0): Testing framework for running the test suite
- **Sphinx** (>=4.0): Documentation generation (for building these docs)
- **sphinx-rtd-theme**: ReadTheDocs theme for Sphinx documentation

System Requirements
====================

- **Disk Space**: ~50 MB for source code and dependencies
- **RAM**: 512 MB minimum for typical grid generation tasks
- **CPU**: Any modern processor (grids are typically computed in seconds)

Troubleshooting
===============

ImportError: No module named 'numpy'
------------------------------------

If you get an error about missing NumPy, install it first:

.. code-block:: bash

    pip install numpy

Could not find version of package molgrid
------------------------------------------

Make sure you're installing from a directory with a valid setup.py file:

.. code-block:: bash

    cd /path/to/molgrid
    pip install -e .

Development Installation
=========================

For development, create a virtual environment and install in editable mode:

.. code-block:: bash

    python -m venv venv
    source venv/bin/activate  # On Windows: venv\Scripts\activate
    pip install -e ".[test]"

This allows you to modify the source code and immediately see changes without reinstalling.
