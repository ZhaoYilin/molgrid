molgrid.prune

class molgrid.prune.Becke(molecule)[source]

Bases: object

Classes

Becke(molecule)

Overview

The Becke class implements Becke’s partitioning scheme for molecular integration grids. This scheme assigns weights to grid points based on distances to atomic nuclei, enabling atom-specific contributions to the molecular grid.

Key Concepts

Voronoi Cells: Each atom gets a Voronoi cell proportional to its size (Bragg or van der Waals radius).

Weight Function: For a point \(r\), the weight is:

\[w_i(r) = \frac{\rho_i(r)}{\sum_j \rho_j(r)}\]

where \(\rho_i(r)\) is a smoothed Voronoi cell indicator for atom \(i\).

Smoothing: Polynomial smoothing ensures smooth transitions between cells:

\[\begin{split}s(\mu) = \begin{cases} 0 & \mu \leq -1 \\ \mu - \mu^3/3 & -1 < \mu < 1 \\ 1 & \mu \geq 1 \end{cases}\end{split}\]

Key Methods

  • _weight_function(): Compute Becke weights for grid points

  • _voronoi_polyhedron(): Compute raw Voronoi indicators

  • _smoothing_function(): Apply polynomial smoothing

Example Usage

from molgrid import MolecularGrid, Molecule

mol = Molecule([
    ['O', 0.0, 0.0, 0.0],
    ['H', 0.96, 0.0, 0.0],
])

mol_grid = MolecularGrid(mol)

# Apply Becke partitioning
mol_grid.apply_becke_weights()

# Get atom-specific weights
weights = mol_grid.get_full_weights()       # (N,) array
grid_with_weights = mol_grid.get_molecular_grid_with_becke()  # (N, 4) array

Becke Parameters

  • do_becke_hetero (bool): Whether to use elemental radii adjustment for heteroatomic molecules (default: True)

When do_becke_hetero=True, the effective radius for each atom is modified based on electronegativity differences to better distribute weight in heteronuclear systems.

Mathematical Details

Distance Metric: Uses normalized distances

\[d_{ij} = \frac{r_{ij}}{r_{ij}^{\text{cov}} + r_{ij}^{\text{cov}}}\]

where \(r_{ij}^{\text{cov}}\) is the sum of covalent radii.

Voronoi Indicator:

\[\mu_i(r) = \prod_{j \neq i} \text{smooth}(d_{ij}(r) - d_{ik}(r))\]

Special Cases

Homonuclear Molecules: All atoms have equal size, producing symmetric Voronoi partitions.

# H2 molecule - equal weights everywhere
h2 = Molecule([['H', 0, 0, 0], ['H', 0.74, 0, 0]])
grid = MolecularGrid(h2)
grid.apply_becke_weights()

Heteroatomic Molecules: Radii adjusted for different elements.

# OH molecule - O gets larger share near nuclei
oh = Molecule([['O', 0, 0, 0], ['H', 0.96, 0, 0]])
grid = MolecularGrid(oh)
grid.apply_becke_weights(do_becke_hetero=True)
