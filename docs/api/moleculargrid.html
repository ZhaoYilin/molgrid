Build a molecular integration grid by combining atomic grids.

Classes

MolecularGrid(atoms[, nshells, nangpts])

Build a molecular integration grid by combining atomic grids.

Overview

The MolecularGrid class combines atomic grids from multiple atoms into a single molecular integration grid. It supports:

  • Grid combination with flexible atom ordering

  • Becke partitioning for atom-specific weights

  • Grid pruning to remove low-weight points

  • Direct molecular grid generation from Molecule objects

Key Methods

  • get_full_grid(): Returns combined grid points

  • get_full_weights(): Returns combined weights

  • get_full_grid_with_weights(): Returns points and weights as 4-column array

  • apply_becke_weights(): Apply Becke partitioning

  • get_molecular_grid_with_becke(): Get Becke-weighted grid

  • prune_grid(): Remove low-weight points

Parameters

  • atoms (list or Molecule): List of [symbol, x, y, z] or Molecule object

  • nshells (int, optional): Number of radial shells per atom (default: 32)

  • nangpts (int, optional): Number of angular Lebedev points per atom (default: 110)

Properties

  • natom (int): Number of atoms in the molecular grid

  • atomic_grids (list): List of constituent AtomicGrid objects

  • grid_weights (ndarray): Current weights for all grid points

Example Usage

from molgrid import Molecule, MolecularGrid

# Create water molecule
mol = Molecule([
    ['O', 0.0, 0.0, 0.0],
    ['H', 0.9572, 0.0, 0.0],
    ['H', -0.2390, 0.9265, 0.0],
])

# Generate molecular grid
mol_grid = MolecularGrid(mol, nshells=32, nangpts=110)

# Get raw grid data
points = mol_grid.get_full_grid()      # Shape: (N, 3)
weights = mol_grid.get_full_weights()  # Shape: (N,)

# Apply Becke partitioning
mol_grid.apply_becke_weights()
becke_weights = mol_grid.get_full_weights()

# Prune low-weight points
mol_grid.prune_grid(threshold=1e-6)
pruned_points = mol_grid.get_full_grid()
print(f"Reduced from {len(points)} to {len(pruned_points)} points")

Grid Combination Strategy

Points from atomic grids are combined in atom order:

# If atoms are [O, H, H] with grid sizes [3500, 2200, 2200]
# Combined grid has 7900 points with ordering:
# - Points 0-3499: Oxygen grid
# - Points 3500-5699: First Hydrogen grid
# - Points 5700-7899: Second Hydrogen grid

Becke Partitioning

After calling apply_becke_weights(), weights are normalized across all atoms:

\[w_i^{\text{Becke}} = \frac{w_i^{\text{raw}}}{\sum_j w_j^{\text{raw}}}\]

where raw weights come from spherical atoms with adjustable smoothing.

Grid Pruning

Pruning removes points where Becke weights fall below a threshold:

mol_grid.apply_becke_weights()
mol_grid.prune_grid(method='becke', threshold=1e-6)

# Remaining points have becke_weight >= 1e-6
