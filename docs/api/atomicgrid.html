Overview

The AtomicGrid class generates single-atom spherical integration grids by combining:

  • Radial quadrature: Gauss-Chebyshev Rule on [0, ∞)

  • Angular quadrature: Lebedev Rule on unit sphere

This produces a product grid suitable for integration of spherically symmetric or smooth functions.

Key Methods

  • get_full_grid(): Returns grid points in Cartesian coordinates

  • get_full_weights(): Returns integration weights

  • get_radii(): Returns radial distances from center

  • get_angular_grid(): Returns angular grid points and weights

Parameters

  • atomic_symbol (str): Chemical element symbol (e.g., ‘H’, ‘C’, ‘N’, ‘O’)

  • nshells (int, optional): Number of radial shells (default: 32)

  • nangpts (int, optional): Number of angular Lebedev points (default: 110)

Example Usage

from molgrid import AtomicGrid
import numpy as np

# Create hydrogen atomic grid
h_grid = AtomicGrid('H', nshells=32, nangpts=110)

# Get grid information
points = h_grid.get_full_grid()      # Shape: (N, 3)
weights = h_grid.get_full_weights()  # Shape: (N,)
radii = h_grid.get_radii()          # Shape: (N,)

print(f"Total points: {len(points)}")
print(f"Weight sum: {weights.sum()}")

# Grid centered at origin
assert np.allclose(h_grid.center, [0, 0, 0])

Grid Sizes

The grid size is determined by:

\[N = n_{\text{shells}} \times n_{\text{angular}}\]

Common Combinations:

  • nshells=20, nangpts=50: ~1000 points (coarse)

  • nshells=32, nangpts=110: ~3500 points (medium)

  • nshells=50, nangpts=170: ~8500 points (fine)
